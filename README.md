## Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites

None.

### Installation

1. Clone the repo
   ```bash
   git clone https://github.com/Nitish-ku/portfolio.git
   ```
2. Navigate to the project directory
   ```bash
   cd your-portfolio
   ```
3. Open `dist/index.html` in your browser.
